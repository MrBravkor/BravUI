# BravUI

Addon World of Warcraft (Retail) — Remplacement complet de l'interface.
Auteur : Bravkor

## Addons
- **BravUI** — Core principal
- **BravUI_Lib** — Bibliothèque partagée
- **BravUI_Menu** — Interface de configuration

## Installation
Télécharger via CurseForge ou GitHub Releases.
Placer les dossiers dans `World of Warcraft/Interface/AddOns/`.
